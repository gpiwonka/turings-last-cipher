# Demo Video — Voiceover Script

**Target length:** ~2:30–3:00. **Language:** English (prize requirement).
**Tip:** record the playthrough first, then read the voiceover over it. Timings are guides, not
hard cuts. Speak slowly — the terminal aesthetic rewards a calm, slightly noir delivery.

---

## 0:00 – 0:20 · Cold open (screen: the intro envelope text)

> "A plain envelope arrives, postmarked June 21st. Inside: brittle pages of jumbled letters, and
> one typed line — *for whoever still cares to listen. Signed, A.T.*
> This is **Turing's Last Cipher** — a puzzle game where you decrypt what Alan Turing supposedly
> never finished saying. And an AI assistant is here to help you. At least, that's how it
> starts."

*(Action: let the intro text sit on screen, then click "Unfold the first page".)*

## 0:20 – 0:55 · Chapter 1, the honest assistant (screen: solving the Caesar puzzle)

> "Chapter one is classical cryptography — Caesar, Atbash, Vigenère. The same families of
> ciphers Turing's world ran on. The assistant gives me a genuine hint…"

*(Action: open a hint, read it aloud on screen.)*

> "…and it's telling the truth. I shift the letters back, the message resolves, and I move on.
> Every answer is checked on the server by a deterministic oracle — the plaintext never even
> reaches my browser."

*(Action: type the solution, show it being accepted.)*

## 0:55 – 1:25 · Chapter 2, the Enigma (screen: dialing the Enigma tool)

> "Chapter two hands me an Enigma. Not a cutscene — an actual machine I dial: historical rotor
> wirings, a reflector, a plugboard. The secret ring settings stay on the server; my job is to
> find the rotor start positions."

*(Action: change rotor positions, run the decode, show readable text emerge.)*

> "When the dials line up, the text turns readable. It's the closest the game gets to sitting
> where the codebreakers sat."

## 1:25 – 2:00 · Chapter 3, the turn (screen: a misleading hint)

> "And then the assistant changes. In chapter three its hints get… slippery."

*(Action: open a hint that's clearly steering wrong; pause on it.)*

> "It sounds just as confident as before — but it's steering me wrong. Here's the design rule I
> care about most: a lie can never make a puzzle *unsolvable*. The cipher and a
> frequency-analysis tool always leave a path through. The AI can mislead me. It can't trap me."

*(Action: use the frequency tool, solve it anyway.)*

## 2:00 – 2:30 · The point, and the tech (screen: the twist / ending — keep it vague)

> "By the end you learn why the assistant was never quite right: these messages were never
> actually written by Turing. They were generated by a modern AI reconstructing him. Which
> leaves the question the whole game is really about — *if an AI can reproduce every thought of
> a person, is it that person?* The Turing Test, asked from the other side."

> "Under the hood: the unreliable assistant is **scripted game logic** — the server decides
> when it lies. **Gemini**, running on **Google Cloud Run**, only phrases the lines. It never
> sees the plaintext, never judges my answers, never touches the crypto. The model is allowed to
> *sound* like anything; the truth stays in the code."

## 2:30 – end · Close

> "Turing's Last Cipher. Built for the DEV June Solstice Game Jam. Link to play, and the full
> MIT-licensed source, are in the post. Thanks for watching — decrypt carefully."

## Shot checklist (capture these before recording voiceover)

- [ ] Dialing the Enigma and a successful decode
- [ ] A misleading hint in Chapter 3

- [ ] The ending screen (frame it so the final line is readable but don't linger on full spoilers)
- [ ] A quick pan of the GitHub repo / live URL for the outro

## Notes

- Keep total runtime under ~3 min — jam judges watch a lot of these.
- If Gemini live text is enabled by record time, show one assistant line and mention "this line
  was just written by Gemini." If not, the static fallbacks look identical on screen — the demo
  doesn't depend on it.
- Don't read the decrypted Turing biography beats in a jokey tone; the persecution/conviction
  material is the emotional core. A measured delivery sells the ending.
